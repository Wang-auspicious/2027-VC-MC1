# VAST MC1 Final Submission Package Design

Date: 2026-07-23
Status: authorized for direct delivery

## Goal

Turn the approved continuous evidence atlas into the complete English competition package: an offline interactive program, a formal HTML answer report, a real interaction walkthrough under four minutes, standalone answers, and a verified ZIP below the PCS size limit.

## Delivery architecture

The English application is a deterministic mirror of `visual_workspace_cn`. It keeps the chat workspace as the entry, preserves the 912-message graph and 3D view, and retains the Q3 → Q2 → Q1 → IGC → prevention narrative. Source messages remain in English. All interface copy is English and all files work offline through relative paths.

The report follows the official answer-sheet field order. It leads with the three challenge answers and uses five captured views from the current atlas: the full message field, Q3 warning almanac, Q2 duty/3D comparison, Q1 evidence tracks, and prevention counterfactual. IGC appears only after the official questions as a bounded synthesis.

The video is a 16:9 screen recording of the English application with six timed segments: data context, Q3 warning, Q2 role migration, Q1 evidence boundary, IGC synthesis, and prevention. Narration states only supported claims. Total duration must remain below 240 seconds.

## Evidence boundary

- The public disclosure actions are observed.
- Pre-release authorization is asserted in dialogue and lacks an independent CivicLoom artifact in the dataset.
- No 17:xx `official_post` is observed.
- The supported diagnosis is control failure with role and channel migration.
- The submission does not assign final personal culpability or claim malicious intent.
- IGC is presented through evidence ranges and observational proxies, never as a validated universal benchmark score.

## Verification

- Every relative link resolves inside the extracted package.
- The English app contains no Chinese interface copy and opens offline.
- All key message IDs and all 912 source messages remain addressable.
- Q3, Q2, Q1, 3D, IGC, and prevention interactions pass browser smoke checks.
- Report screenshots are legible at desktop width and correspond to the current app.
- The video has H.264 video, AAC audio, 1440×900 or 1920×1080 output, and duration below 240 seconds.
- The final ZIP is below 50 MB and contains only submission materials.
